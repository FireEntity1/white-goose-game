# White Goose Game

Play in your browser here! https://fire-entity.itch.io/white-goose-game

## Controls:
  AD/Left-Right - Move
  
  Space/W/Up - Jump
  
  Ctrl/S/Down - Ground slam

A basic 2D platformer and a part of the Goose Game series!
